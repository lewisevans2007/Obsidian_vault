In [[C]] formatted output is a way of printing to the console in a formatted way.

## Usage

```c
// printf is a function that prints to 
// the console and that supports formatting
printf("Hello, %s!\n", "world");
```

## All formats

- `%u` - unsigned int
- `%d` - int
- `%c` - char
- `%s` - string
- `%f` - float
- `%lf` - double
- `%p` - pointer
- `%lu` - unsigned long
- `%lld` - long long
- `%llu` - unsigned long long
- `%hhd` - signed char
- `%hhu` - unsigned char
- `%hd` - short
- `%hu` - unsigned short
- `%ld` - long
- `%lu` - unsigned long
- `%lld` - long long
- `%llu` - unsigned long long
